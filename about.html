Sobre mim

Olá! Meu nome é Leandro, sou apaixonado por tecnologia, design e tudo que envolve criatividade digital. Atualmente estou me aprofundando em desenvolvimento web, explorando HTML, CSS e outras ferramentas que me permitem transformar ideias em interfaces funcionais e atraentes. Este projeto é uma amostra do que venho aprendendo e criando com dedicação e curiosidade.

Este site foi desenvolvido como parte do meu portfólio para compartilhar meus conhecimentos e evolução como desenvolvedor. Cada detalhe — desde os efeitos visuais até a estrutura do código — foi pensado para refletir minha identidade e estilo. A ideia é mostrar que, mesmo com recursos simples, é possível criar experiências digitais marcantes. E claro, tudo isso está disponível no meu GitHub para quem quiser acompanhar ou colaborar!
